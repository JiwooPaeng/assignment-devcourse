# 컴파일 언어 vs 인터프리터 언어
- 컴파일 언어 
  - 사람이 작성한 코드를 기계어로 변환하는 과정(pre-processing)과 기계에서 실행하는 과정(processing)이 존재
  - 실행 단계에서 번역된 컴파일 언어를 사용해 인터프리터 언어보다 실행 속도가 빠르다
  - Rust
- 인터프리터 언어
  - 사람이 작성한 코드를 기계어로 변환하는 동시에 바로 실행(processing 과정만 진행)
  - 인터프리터 언어에 비해 기계어 변환 단계가 필요해 실행이 느리다
  - JavaScript
